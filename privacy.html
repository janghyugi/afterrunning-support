뛴김에 개인정보처리방침

시행일: 2026-04-16

뛴김에는 사용자가 선택한 사진과 입력한 러닝 기록을 바탕으로 이미지를 생성하는 기능을 제공합니다.

1. 처리하는 정보

사용자가 선택한 사진, 직접 입력한 러닝 기록, 선택한 Strava 기록 이미지를 앱 기능 수행을 위해 처리할 수 있습니다.

2. 이용 목적

결과 이미지를 생성하고 저장하기 위한 목적입니다.

3. 처리 방식

사용자가 선택한 사진과 입력한 기록은 앱 기능 수행을 위해 기기 내에서 처리됩니다.

4. 외부 전송 및 저장

앱은 사용자의 사진, 기록, Strava 이미지 데이터를 외부 서버에 업로드하거나 저장하지 않습니다.

5. 회원가입 및 로그인

앱은 별도의 회원가입 또는 로그인 기능을 제공하지 않습니다.

6. 광고 및 추적

앱은 광고 추적을 위한 개인정보를 수집하지 않습니다.
